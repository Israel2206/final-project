# final-project
Projeto Final – Curso de Java (Carga Horária: 40h)

Descrição:
Este projeto final faz parte do curso de Java com carga horária de 40 horas. Ele tem como objetivo aplicar os conhecimentos adquiridos durante o curso, incluindo conceitos de Programação Orientada a Objetos (POO), estruturas de controle, manipulação de dados, arrays, listas, e interação com o usuário via console.

Funcionalidades principais:

Criação e manipulação de classes e objetos;

Definição de atributos e métodos para cada classe;

Uso de herança e encapsulamento;

Implementação de lógica condicional para ações do usuário;

Interação básica com o usuário através do console;

Registro e visualização de dados de forma organizada.

Tecnologias utilizadas:

Java SE (versão X ou superior);

Conceitos de POO: classes, objetos, herança, encapsulamento e polimorfismo;

Estruturas de repetição e decisão (for, while, if/else).

Objetivo do projeto:
Demonstrar a capacidade de criar um sistema funcional em Java aplicando os conceitos aprendidos durante o curso, desenvolvendo lógica de programação, organização de código e boas práticas de desenvolvimento.
